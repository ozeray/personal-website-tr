Scheduled Navigation Management Portal

The portal allows a fully automated process of application, approval and payment for official certification of scheduled maritime navigation companies across Turkey. It also provides various administrative options for map tracking, charts, statistics, dynamic reporting, logging, authentication and authorization. It supports Turkish and English.
I performed full-stack architecture, design, implementation, maintenance and continuous integration of this portal.
I can give more details on demand.

Microservices and Microfrontend Architecture

Microservices and microfrontend architecture as an alternative to the legacy application at Fonet, my former company.
I designed, implemented and documented this framework using Java, Spring Boot, Spring Cloud, Svelte, node.js and other libraries. I leveraged several modern techniques and patterns in this project.
I can give more details on demand.

Authorization Portal

Roles, permissions and departments management portal for the Ministry of Transportation and Infrastructure, Turkish Republic.
The portal leverages lots of rich UI components, such as wizards, accordions, context menus, data tables, popups, etc. to ensure simplicity, usability and flexibility. It supports Turkish and English languages.
I performed full-stack architecture, design, implementation, maintenance and continuous integration of this portal.
I can give more details on demand.

Mapty - Manage Your Daily Workouts

This was a challenge project for Udemy course: The Complete JavaScript.
In this project we used HTML, CSS, Web APIs, JavaScript and leaflet.js to build a daily workout management application. The application detects your current geo-location and allows you to add daily running or cycling workouts on the map. You can navigate through your past workouts on the map, sort, edit or delete them. You can also fit your map to cover existing workouts.
You may try it out for your own.

Pig Game

This was a challenge project for Udemy course: The Complete JavaScript. I took the course to keep up with modern JavaScript features and refresh my JavaScript skills.
In this project we used HTML, CSS and JavaScript to build the so-called pig game. Two players roll dice to accumulate scores and the player who first reaches a score of 50 wins the game.
You may try it out with your friends :)
